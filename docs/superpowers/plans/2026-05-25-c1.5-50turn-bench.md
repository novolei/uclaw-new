# C1.5 — 50-Turn Refactor Benchmark Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a hybrid benchmark that measures the real token / round-trip / cache / cost deltas of the Dirac Phase A/B borrows on an 8-file rename refactor, producing the M2 DoD numbers the closeout reports flagged as MODELED-not-measured.

**Architecture:** A **deterministic replay** path reconstructs the exact LLM request (tool registry + composed system prompt + a message history grown by replaying a fixed golden tool-call sequence) and sums per-turn token components via the app's own `estimate_tokens` — run on a pre-Dirac build (git worktree @ f6447a71) and the post build, then diffed. A **live** path reuses the existing non-Wry `agent::headless` runner against real deepseek/Anthropic providers to confirm the LLM actually adopts the borrows + to measure cache-hit. An orchestration script ties the two builds together and renders the report. Bench-only — no shipping runtime change.

**Tech Stack:** Rust (`tokio`, `async_trait`, `serde_json`), the existing `LlmProvider` trait + `agent::headless` runner + `estimate_tokens`, a bash orchestration script.

**Spec:** `docs/superpowers/specs/2026-05-25-c1.5-50turn-bench-design.md`

```rust
/// Replay a golden sequence, growing the message history and summing the
/// per-turn request token breakdown. No network, no dispatcher, no Wry.
pub fn replay(fixture_dir: &std::path::Path, golden: &str) -> ReplayReport {
    let workspace = fixture_dir.join("workspace");
    let records = load_golden(&fixture_dir.join(format!("{golden}_dirac.jsonl")));
    let tools = bench_tool_definitions(&workspace);
    let tool_def_tokens = estimate_tokens(&serde_json::to_string(&tools).unwrap());
    // LCD system prompt (exists in both builds). Mode = the default ACT mode.
    let system_prompt = crate::agent::mode_prompts::compose_system_prompt(
        "", Some(&workspace), &crate::agent::types::SafetyMode::default());
    let system_prompt_tokens = estimate_tokens(&system_prompt);

    let mut history: Vec<ChatMessage> = vec![ChatMessage {
        role: MessageRole::User,
        content: vec![ContentBlock::Text { text: "Rename old_name to new_name across the workspace.".into() }],
        compacted: false,
    }];
    let mut per_turn = Vec::new();
    let mut round_trips = 0usize;

    for rec in &records {
        // Measure the request that WOULD be sent THIS turn.
        let message_tokens = estimate_tokens(&serde_json::to_string(&history).unwrap());
        per_turn.push(TurnBreakdown {
            turn: rec.turn,
            system_prompt_tokens,
            tool_def_tokens,
            message_tokens,
            total: system_prompt_tokens + tool_def_tokens + message_tokens,
        });
        if rec.tool_name == "__final__" {
            history.push(ChatMessage { role: MessageRole::Assistant,
                content: vec![ContentBlock::Text { text: rec.tool_args["text"].as_str().unwrap_or("").into() }],
                compacted: false });
            break;
        }
        round_trips += 1;
        // assistant tool_use turn + user tool_result turn (grows history like the real loop)
        history.push(ChatMessage { role: MessageRole::Assistant, content: vec![ContentBlock::ToolUse {
            id: format!("call_{}", rec.turn), name: rec.tool_name.clone(), input: rec.tool_args.clone() }], compacted: false });
        history.push(ChatMessage { role: MessageRole::User, content: vec![ContentBlock::ToolResult {
            tool_use_id: format!("call_{}", rec.turn), content: rec.mock_tool_result.clone(), is_error: Some(false) }], compacted: false });
    }

    ReplayReport {
        fixture: fixture_dir.file_name().unwrap().to_string_lossy().into(),
        golden: golden.into(),
        turns: per_turn.len(),
        round_trips,
        total_input_tokens: per_turn.iter().map(|t| t.total).sum(),
        per_turn,
    }
}
```
